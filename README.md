
# Movie Introduce App

## TheMoviedb API 를 이용한 영화소개 어플

 ### 사용한 라이브러리
 * React
  * Typescript
  * React-query
  * Hooks
  * react-hook-form
  * Styled-compotnents
  * Framer-motion 
### 주요기능
  1.영화에대한정보를보
  2.예고편을볼수있는 비디오가 재생
  3.영화검색
